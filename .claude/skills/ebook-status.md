# /ebook-status — 進捗確認

指定した書籍の進捗状況を表示する。
引数: 書籍スラッグ（例: `/ebook-status ai-business-guide`）

## 手順

1. 引数のスラッグに対応する `books/{slug}/book.yaml` を読み込む
2. `books/{slug}/chapters/` 内の全ファイルを走査する
3. 各章のフロントマターから `status`、`word_count`、`reviewed_by` を取得する
4. 進捗サマリーを以下の形式で表示する

## 出力形式

```
{書籍タイトル}
著者: {著者名} | ステータス: {全体ステータス}

章              | ステータス | 文字数 | レビュー済み
--------------- | ---------- | ------ | ------------
はじめに         | complete   | 1,200  | 校正,校閲,編集長
第1章: ...      | writing    | 3,400  | -
第2章: ...      | draft      | 0      | -
...
あとがき         | draft      | 0      | -

進捗: 2/10章 完了
総文字数: 4,600 / 目標 50,000文字 (9.2%)

次のステップ: /ebook-write {slug} 2
```

## ステータス表記
- draft — 未着手
- writing — 執筆中
- review — レビュー済み
- complete — 完了
