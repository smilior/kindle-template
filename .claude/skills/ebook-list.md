# /ebook-list — 管理中の書籍一覧

管理中の全書籍の一覧と概要を表示する。

## 手順

1. `books/` ディレクトリ内のサブディレクトリを走査する
2. 各ディレクトリの `book.yaml` を読み込む
3. 以下の形式で一覧を表示する

## 出力形式

```
管理中の書籍一覧

スラッグ              | タイトル                | ステータス | 章 | 文字数
--------------------- | ----------------------- | ---------- | -- | ------
ai-business-guide     | AIビジネス入門           | in_progress | 4/10 | 18,200
side-hustle-101       | 副業の教科書             | draft       | 0/8  | 0
python-data-science   | Pythonデータ分析入門     | complete    | 12/12 | 52,300

合計: 3冊
```

## 書籍がない場合

```
管理中の書籍はありません。
/ebook-init で新しい書籍プロジェクトを作成してください。
```
