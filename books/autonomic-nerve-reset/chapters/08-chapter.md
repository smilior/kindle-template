# 第8章　【Week4】Day22〜Day30 — 習慣を定着させ、ストレス耐性を高める

いよいよ最終週です。

ここまで3週間、呼吸・睡眠・生活リズムと、自律神経を支える3本柱を一つずつ積み上げてきました。

Week4のテーマは2つあります。

1つ目は、**これまでの習慣を「定着」させること**。
2つ目は、**ストレスへの対処力を高めること**。

「せっかく続けてきたのに、また元に戻ったらどうしよう」。
そんな不安を抱えている方もいるかもしれません。

大丈夫です。
この1週間で、習慣を「頑張って続けるもの」から「自然にやるもの」へと変えていきましょう。

## Day22〜23：マインドフルネス入門 — 「今ここ」に集中する1分間瞑想

**マインドフルネス**とは、「今この瞬間に意識を向けること」です。

難しそうに聞こえますが、やることはとてもシンプル。
**1分間、呼吸に意識を集中するだけ**です。

具体的な手順はこうです。

1. 椅子に座り、背筋を軽く伸ばす
2. 目を閉じるか、視線を斜め下に落とす
3. 鼻からゆっくり息を吸い、口からゆっくり吐く
4. 「吸っている」「吐いている」と心の中で実況する
5. 雑念が浮かんだら、気づいた時点で呼吸に戻す

ポイントは、**雑念が浮かんでも自分を責めないこと**です。

「あ、仕事のことを考えていたな」と気づけたら、それだけで十分。
むしろ「気づけた」こと自体がトレーニングになっています。

Week1で身につけた呼吸法と組み合わせると、さらに効果的です。
4-7-8呼吸をしながら「吸って……止めて……吐いて……」と実況する。
これだけで、自律神経がスッと落ち着くのを感じられるはずです。

Day22はまず1分間から。
Day23には2分間に伸ばしてみましょう。

おすすめのタイミングは、**朝の光浴びのあと**か**寝る前のルーティンの中**です。
すでに習慣化している行動にくっつけると、自然と定着しやすくなります。

## Day24〜25：ストレス対処の引き出しを増やす

自律神経を乱す最大の敵は、**慢性的なストレス**です。

ストレスそのものをゼロにすることはできません。
でも、「ストレスを受けたときの対処法」を持っておくことはできます。

ここでは**3つの選択肢**を紹介します。

**1. 書く**

モヤモヤしたとき、頭の中にあることを紙に書き出す方法です。
ノートでもスマホのメモでも構いません。

「何にイライラしているのか」を言葉にするだけで、気持ちが整理されます。
これは**エクスプレッシブ・ライティング**と呼ばれる手法で、ストレスホルモンの低下が研究で確認されています。

3分間、思いつくままに書くだけでOKです。

**2. 話す**

信頼できる人に話を聞いてもらうことも、強力なストレス解消法です。

ただし、「解決策がほしい」のか「ただ聞いてほしい」のかを自分で把握しておくのがコツ。
「ちょっと愚痴を聞いてほしいんだけど」と先に伝えるだけで、お互いにラクになります。

**3. 動く**

体を動かすと、ストレスホルモンが分解されやすくなります。

激しい運動でなくて大丈夫です。
5分間の散歩、階段の上り下り、その場で軽くスクワット。
Day17〜18で練習した「プチリセット」がここでも活躍します。

Day24に、この3つの方法をそれぞれ試してみてください。
Day25には、自分に合った方法を**「ストレス対処カード」**として書き出しておきましょう。

「イライラしたら→5分散歩」「不安を感じたら→3分間メモ」。
こんなふうに、**あらかじめ決めておく**ことが大切です。

ストレスの渦中にいると、冷静な判断はできません。
だからこそ、元気なうちに対処法を「カード」として用意しておくのです。

## Day26〜27：「サボった日」のリカバリー法

ここで大切な話をさせてください。

**30日間、一度も休まず完璧にこなす必要はありません。**

むしろ、「サボった日」がないほうが不自然です。
仕事が忙しい日もある。体調が悪い日もある。気分が乗らない日もある。

問題は「サボったこと」ではなく、**「サボったことで自分を責めて、やめてしまうこと」**です。

これを**「どうにでもなれ効果」**と呼びます。
ダイエットで1日食べすぎると「もういいや」と投げ出してしまう、あの心理です。

リカバリーの方法はシンプルです。

**翌日、何か1つだけやる。**

呼吸法でも、寝る前のストレッチでも、朝の光浴びでも。
30あるプログラムの中から、一番ラクにできるものを1つだけ。

それだけで「続いている自分」を取り戻せます。

Day26〜27は、あえて**「完璧を手放す練習」**の日です。

これまでのプログラムを振り返り、「できなかった日」がどれくらいあったか数えてみてください。
そして、**それでも今日ここまで読み進めている自分を認めてあげてください**。

80%できていれば、十分すぎるほどの成果です。

## Day28〜29：自分だけの「リセットルーティン」を完成させる

さあ、仕上げに入ります。

3週間で学んだテクニックの中から、**あなた自身に合ったものを選んでオリジナルルーティンを作りましょう**。

以下のテンプレートを参考にしてください。

**朝のルーティン（5分）**
- [ ] 起床後、窓際で深呼吸3回
- [ ] 朝の光を15分浴びる（通勤でもOK）
- [ ] 朝食を決まった時間に摂る

**日中のリセット（2分）**
- [ ] 昼休みに5分間散歩
- [ ] 午後の眠気がきたらため息リセット
- [ ] ストレスを感じたら対処カードを見る

**夜のルーティン（10分）**
- [ ] 就寝90分前にスマホを手放す
- [ ] 38〜40度のお風呂またはシャワー
- [ ] 寝る前に1分間マインドフルネス
- [ ] 4-7-8呼吸法で入眠

全部やる必要はありません。
朝・昼・夜からそれぞれ**1〜2つずつ選ぶだけ**で十分です。

Day28に仮のルーティンを組み立て、Day29に実際にやってみる。
違和感があれば入れ替える。

**「これなら毎日できそう」と思えるラインを見つけること**が、このDay28〜29の目標です。

30日前の自分と今の自分を比べてみましょう。

- [ ] 朝の目覚めが以前よりラクになった

5つ以上なら、素晴らしい成果です。
周りの人から「最近元気だね」と言われているかもしれません。

全部にチェックがつかなくても、**まったく問題ありません**。
1つでも変化があれば、それはあなたが行動した証拠です。
